## Numpy数组的运算

- 大小相等的数组之间的任何算术运算都会将运算应用到元素级
- 数组与标量的算术运算会将标量值传播到各个元素
- 大小相同的数组之间的比较会生成布尔值数组
- numpy对切片的修改会反映在原数组身上
